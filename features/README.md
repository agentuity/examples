# Feature Examples

Examples showcasing Agentuity platform capabilities: cron jobs, KV storage, durable streams, and more.

| Example | Features | Description |
|---------|----------|-------------|
| [scheduled-digest](./scheduled-digest) | Cron, KV, Streams | Cron-powered content digest published to shareable URLs |

## Getting Started

```bash
cd scheduled-digest
bun install
bun run dev
```

Each example includes setup instructions in its README.
