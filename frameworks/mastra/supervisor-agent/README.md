## ğŸ—ï¸ Architecture

This example consists of three agents:

1. **Copywriter Agent** - Writes initial blog post content based on a given topic
2. **Editor Agent** - Refines and edits the content to improve quality and readability
3. **Publisher Agent (Supervisor)** - Coordinates the entire process by calling the copywriter first, then the editor

The supervisor agent uses Mastra's tool system to orchestrate the workflow between specialized agents.
